# 🎯 HH.ru Vacancy Parser Bot

![Python Version](https://img.shields.io/badge/Python-3.10+-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Aiogram](https://img.shields.io/badge/Aiogram-3.x-2CA5E0?style=for-the-badge&logo=telegram&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=for-the-badge&logo=sqlite&logoColor=white)

Асинхронный Telegram-бот для мониторинга свежих вакансий Junior Data Science с площадки hh.ru. Бот оснащен "умной" системой обхода защиты: сначала используется быстрый JSON-парсер, а в случае блокировки — полноценная симуляция браузера с визуальной диагностикой.

## ✨ Ключевые возможности

*   **Двухуровневый парсинг:** 
    1. `curl_cffi` — имитация TLS-отпечатка Chrome для быстрых запросов.
    2. `Playwright (Chromium)` — тяжелая артиллерия: эмуляция реального пользователя, если легкий способ заблокирован.
*   **Визуальная диагностика:** При возникновении проблем с поиском бот автоматически делает скриншот страницы (`last_attempt.png`), чтобы вы могли увидеть причину (капча, блок или пустая выдача).
*   **Адаптированны прокси:** Логика адаптирована для работы через прокси как для бота так и для парсера (для парсера необходим строго IPv4 прокси т.к. hh.ru не поддерживает IPv6).
*   **Smart Deduplication:** База данных на `aiosqlite` предотвращает повторную отправку вакансий и автоматически очищается от мусора.

## 🛠 Требования

* **Python** 3.10 или новее
* Токен Telegram-бота (получить у [@BotFather](https://t.me/BotFather))
* Ваш Telegram ID (получить у [@userinfobot](https://t.me/userinfobot))
* Прокси для бота (необходимо, чтобы бот мог работать на территории РФ)
* Прокси IPv4 для парсера (необходимо если hh.ru блокирует вай ip)

## 🚀 Установка и запуск

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/b4end/parser_hh.git
   cd parser_hh
   ```

2.  **Установите зависимости:**
    ```bash
    python -m venv .venv
    source .venv/bin/activate  # Для Linux/macOS
    # .venv\Scripts\activate   # Для Windows
    pip install -r requirements.txt
    ```

3.  **Установите браузер и системные библиотеки (ВАЖНО):**
    ```bash
    pip install playwright
    playwright install chromium
    # sudo playwright install-deps  # Только для Linux
    ```

4. **Настройте переменные окружения:**
   Скопируйте файл примера и заполните свои данные:
   ```bash
   cp .env.example .env
   ```
   **Пример `.env` файла:**
   ```env
   BOT_TOKEN=123456789:ABCDefgh...
   ADMIN_ID=123456789
   PROXY_URL=http://user:pass@ip:port
   HH_PROXY=http://user:pass@ip:port
   ```

5. **Запустите бота:**
   ```bash
   python bot.py
   ```

## 🗄 Структура проекта

* `bot.py` — Точка входа, инициализация Aiogram, настройка фоновых задач (парсер и очистка БД).
* `parser.py` — Логика HTTP-запросов (`curl_cffi`), парсинг SSR JSON, форматирование данных.
* `database.py` — Асинхронная работа с SQLite, учет отправленных вакансий.
* `bot_database.db` — Файл базы данных (создается автоматически при первом запуске).

## 🖥 Использование
После запуска `bot.py` скрипт начнет работать в фоновом режиме. 
* Напишите боту `/start` в Telegram, чтобы проверить его работоспособность.
* Каждые 5 минут бот будет опрашивать HH.ru и присылать новые вакансии в формате удобных карточек.
* База данных автоматически очищается от записей старше 7 дней раз в сутки.

## 🔍 Отладка
Если бот сообщает, что вакансий не найдено, проверьте файл `last_attempt.png` в корневой папке проекта. Он покажет реальное состояние страницы, которую видит бот.
