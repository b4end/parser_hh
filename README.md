# 🎯 HH.ru Vacancy Parser Bot

![Aiogram](https://img.shields.io/badge/Aiogram-3.x-2CA5E0?style=for-the-badge&logo=telegram&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-003B57?style=for-the-badge&logo=sqlite&logoColor=white)

Асинхронный Telegram-бот для мониторинга свежих вакансий (Data Science / Machine Learning) с площадки HH.ru. Бот написан на Python с использованием `aiogram 3`, работает без тяжелых браузеров (Selenium/Playwright) и отправляет красиво оформленные карточки вакансий прямо в Telegram.

## ✨ Ключевые возможности

* **Обход Anti-Bot систем:** Использование `curl_cffi` позволяет эмулировать TLS-отпечатки реального браузера (Google Chrome), сводя к минимуму риск получения капчи от HH.
* **Фильтрация по стажу:** Строгий отбор вакансий по опыту работы («Без опыта», «От 1 года до 3 лет»). Сеньорские вакансии автоматически игнорируются.
* **Богатые метаданные:** В Telegram отправляется не просто ссылка, а полная карточка: зарплата, формат работы, количество текущих просмотров, количество откликов, бейджи аккредитованной ИТ-компании и возможность работы по ГПХ.
* **Smart Deduplication:** Локальная база данных на SQLite (`aiosqlite`) запоминает отправленные вакансии и не спамит дубликатами. Встроена автоочистка старых записей.

## 🛠 Требования

* **Python** 3.10 или новее
* Токен Telegram-бота (получить у [@BotFather](https://t.me/BotFather))
* Ваш Telegram ID (получить у [@userinfobot](https://t.me/userinfobot))
* Прокси (необходимо, чтобы бот мог работать на территории РФ)

## 🚀 Установка и запуск

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/b4end/parser_hh.git
   cd parser_hh
   ```

2. **Создайте виртуальное окружение и установите зависимости:**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Для Linux/macOS
   # .venv\Scripts\activate   # Для Windows
   pip install -r requirements.txt
   ```

3. **Настройте переменные окружения:**
   Скопируйте файл примера и заполните свои данные:
   ```bash
   cp .env.example .env
   ```
   **Пример `.env` файла:**
   ```env
   BOT_TOKEN=123456789:ABCDefgh...
   ADMIN_ID=123456789
   HH_PROXY=http://user:pass@ip:port
   ```

4. **Запустите бота:**
   ```bash
   python bot.py
   ```

## 🗄 Структура проекта

* `bot.py` — Точка входа, инициализация Aiogram, настройка фоновых задач (парсер и очистка БД).
* `parser.py` — Логика HTTP-запросов (`curl_cffi`), парсинг SSR JSON, форматирование данных.
* `database.py` — Асинхронная работа с SQLite, учет отправленных вакансий.
* `bot_database.db` — Файл базы данных (создается автоматически при первом запуске).

## 🖥 Использование
После запуска `bot.py` скрипт начнет работать в фоновом режиме. 
* Напишите боту `/start` в Telegram, чтобы проверить его работоспособность.
* Каждые 5 минут бот будет опрашивать HH.ru и присылать новые вакансии в формате удобных карточек.
* База данных автоматически очищается от записей старше 7 дней раз в сутки.
